# cloud-itonami-iso3166-che

Open ISO 3166 Blueprint for **CHE**: Switzerland.

Independent public-sector market-entry & procurement-compliance service
for an already-incorporated operator entering public contracts in Switzerland.

## Official surface (web-verified)

- Procurement: simap.ch federal e-procurement platform (WTO GPA member)
- Business/tax: UID/IDE/MWST via Federal Commercial Registry Office / cantonal commercial registers

## What this is NOT

- **Not the government of Switzerland.**
